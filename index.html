Hallo und willkommen zu unserem kleinen HTML/CSS Basics Kurs

Zum gegenwärtigen Zeitpunkt besteht diese Website eher aus Beispieldateien aus meinen Grundlagenvorlesungen zu den Themen HTML, CSS und responsive Design, als aus Seiten mit hilfreichen Informationen. Ich hoffe aber, dass bald mehr direkt auf der Website nachzulesende Inhalte hinzukommen. Bis dahin (und eigentlich auch darüber hinaus), empfehle ich die folgenden beiden eigenen zu dieser Website gehörenden Referenzlinks, um das Thema besser zu verstehen:

  • Ein FigJam Dokument mit Folien und Diagrammen zu ausgewählten Themen aus dem Bereich HTML/CSS/Responsive Design
  • Das GitHub Repository, welches diese Website und Beispieldateien enthält

Ausserdem gibt es natürlich jede Menge Ressourcen im Netz, wo HTML und CSS besser und ausführlicher erklärt werden, als von mir. Die aus meiner Sicht wichtigsten:

  • Selfhtml, Tutorialsite und Referenz zum Thema Webcoding der ersten Stunde. Mit Sicherheit die bekannteste deutschsprachige Website über HTML, CSS und Javascript.
  • developer.mozilla.org — die Leute, die uns Firefox bringen sind imho auch die letzte Instanz, wenn es um Fragen des Webcoding und der Webtechnologie allgemein geht. Falls verschiedene Referenzseiten verschiedende Informationen liefern, würde ich immer dieser hier trauen. In englischer Sprache (auch wenn es vereinzelt auf Deutsch übersetzte Seiten gibt).
  • W3 Schools. Dies ist meiner Meinung nach die beste Website, um sich (Web)Coding unterschiedlichster Art selbst beizubringen. Nicht nur verfügt die Website über eine Unmenge an Referenzsammlungen zu den verschiedenen Computersprachen, als besonders hilfreiche Funktion gibt es interaktive Code Beispiele, die direkt im Browser editiert werden könne, um das Erlernte auszuprobieren. Leider nur auf Englisch verfügbar.

Diese Website ist zu 100% pures gutes altes statisches HTML und "Vanilla" CSS.
